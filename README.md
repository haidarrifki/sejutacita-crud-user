# CRUD USERS & LOGIN WITH REFRESH TOKEN

This project is for assessment purpose only from https://sejutacita.id/

## Pre-requisite
1. Node.js >= v12.x.
2. MongoDB v3.6.3

## Next steps
- clone this repository
- ```yarn install```
- ```yarn dev```
- base url `http://localhost:1337/`
